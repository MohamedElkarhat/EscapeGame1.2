     			 Projet Escape Game


Description du projet :
 
Il s’agit d’un jeu numérique basé sur le concept des Escape Games. 
il ya 3 modes de jeu: 

1) 	le mode challenge :
 		L'intelligence artificielle de l’ordinateur joue le rôle de défenseur. Elle définit une 		combinaison de X chiffres aléatoirement. 
		Le joueur a le rôle d’attaquant, il doit trouver la combinaison secrète.

2)	le mode défenseur:
		Le joueur définit une combinaison de X chiffres aléatoirement.
		L'intelligence artificielle de l’ordinateur doit trouver la combinaison secrète.

3)	le mode Duel: 
		Le joueur et l’intelligence artificielle de l’ordinateur jouent tour à tour. Le premier 		à trouver la combinaison secrète de l'autre a gagné !



Installation :  

JDK 8 et supérieur.
IDE : Eclipse
Log4j.jar
log4j-api 2.13.2.jar
log4j-core-2.13.2.jar


Démarrage

Dans le dossier: src il'ya le package : fr.em.Game.
On  lance l'execution de la classe Game qui est le Main de l'application.
On obtien sur la console le menu de Bienvenue dans le jeux Escape Game . Et les règles de jeu.
